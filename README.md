        <h1>Gerador de Tabuada - C# com Windows Forms</h1>

        <p>
            Este repositório contém um aplicativo simples desenvolvido em <strong>C#</strong> utilizando <strong>Windows Forms</strong> para gerar a tabuada de qualquer número desejado. 
            O gerador de tabuada é uma ferramenta prática e interativa, ideal para estudantes e profissionais que precisam de uma referência rápida para operações de multiplicação.
        </p>

        <h2>Funcionalidades</h2>
        <ul>
            <li><strong>Interface Gráfica (GUI):</strong> Ao executar o programa, uma janela é exibida com campos para inserção de dados e um botão para gerar a tabuada.</li>
            <li><strong>Entrada de Dados:</strong> O usuário insere, por meio de um campo de texto, o número para o qual deseja gerar a tabuada.</li>
            <li><strong>Geração da Tabuada:</strong> Ao clicar no botão, o programa calcula automaticamente e exibe a tabuada do número informado, geralmente de 1 a 10.</li>
            <li><strong>Exibição Formatada:</strong> A tabuada é exibida de forma clara e organizada dentro da interface, facilitando a leitura e o entendimento.</li>
        </ul>

        <h2>Exemplo de Uso</h2>
        <ol>
            <li>Abra o programa.</li>
            <li>Insira o número desejado no campo indicado.</li>
            <li>Clique no botão “Gerar Tabuada”.</li>
            <li>Veja a mágica acontecer: a tabuada será exibida logo abaixo!</li>
        </ol>
